## 📖 About the Original Version (This Repository)

"Suburbious" is a puzzle and management video game originally developed as the final project for the **Programming Paradigms** course. 

The goal of the game is to place different structures on a grid board to score the highest possible points, managing building collapses, Bigfoot invasions, and limited resources.

### 🛠️ Tech Stack
- **Language:** Python 3
- **GUI Framework:** `wxPython` (Desktop Application)
- **Paradigm:** Object-Oriented Programming (OOP)

### ✨ Key Technical Features
* **2D Matrix Logic:** Management of collisions, neighborhoods, and routing within a dynamic grid.
* **Propagation Algorithms:** Recursive functions to detect and execute building "collapses" when 3 or more of the same type are adjacent (Match-3 mechanics).
* **Dynamic UI Rendering:** Dynamic swapping of graphical elements, button state management, and real-time asynchronous events (`wx.Timer`).
